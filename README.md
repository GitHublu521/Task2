# Lab 2: Ablation Study on CIFAR-10

## Overview

In this lab, you will conduct a series of ablation studies on CIFAR-10 to understand the impact of key deep learning design choices: **weight initialization**, **regularization**, **BatchNorm**, **data augmentation for robustness**, and **hyperparameter optimization**.

You will implement specific components (marked with `TODO`) while the experiment infrastructure (training loops, evaluation, plotting) is provided.

## Tasks

### Task 1: Weight Initialization (models/mlp.py)

Implement `_init_weights()` in the `DeepMLP` class to apply Xavier and Kaiming initialization methods. The orchestration code will train a 20-layer MLP with each method and compare activation statistics and training curves.

**You implement:** `models/mlp.py` → `_init_weights()`

### Task 2: Regularization (tasks/task2_regularization.py)

Define 4 regularization configurations (combinations of dropout and weight decay) in `get_reg_configs()`. The orchestration code will train a CNN under each configuration and compare generalization gaps.

**You implement:** `tasks/task2_regularization.py` → `get_reg_configs()`

### Task 3: BatchNorm (models/cnn.py)

Implement `SimpleCNN` and `DeeperCNN` in `models/cnn.py`. The DeeperCNN should conditionally include BatchNorm layers based on the `use_bn` parameter. The orchestration code compares all three configurations.

**You implement:** `models/cnn.py` → `SimpleCNN.__init__()`, `DeeperCNN.__init__()`

### Task 4: Robustness to Distribution Shift (tasks/task4_robustness.py)

Train two SimpleCNN models — one with and one without data augmentation — then evaluate both on **3 CIFAR-10-C corruption types** of your choice. Pick 3 from different categories (e.g., one noise, one blur, one weather/digital). Edit the `"corruptions"` list in `config.json` to select your 3 types (see `"all_corruptions"` for the full list).

**You implement:** `tasks/task4_robustness.py` → training section (marked TODO) + choose 3 corruptions in `config.json`

### Task 5: Hyperparameter Search (tasks/task5_hparam_search.py)

Implement the Optuna objective function to search over learning rate, weight decay, dropout, and batch size.

**You implement:** `tasks/task5_hparam_search.py` → `objective()`
